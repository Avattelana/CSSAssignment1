CSS Properties

  • "text-align" allows you to set the positioning of your code to different options such as left, right, and center.
  • "text-shadow" allows you to create a shadow of color behind your text. You can customize the color and size of the shadow desired.
  • "padding" can add a set amount of space around text, or you can set a different amount of space around the text in the order of top, right, bottom, left.
